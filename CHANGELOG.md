# Changelog

## [0.0.1] - 2026-04-10

### Added
- Compare any file against another Git branch via right-click or command palette
- Automatic `.ts` ↔ `.js` extension fallback when file doesn't exist on target branch
- Configurable path prefix fallbacks (e.g. `src/`) via `file-compare.pathPrefixFallbacks` setting
- Works in remote environments (Docker, SSH, WSL)
